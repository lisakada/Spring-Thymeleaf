About Us

“Good friends, good books, and a sleepy conscience: this is the ideal life. ― Mark Twain ”

MyBookApp is an application that lets the user add a book to the database, view the entire database and search for particular book from the database using the search option. It even allows the user to browse through the ISBN database which is the world's largest book database that gathers data from hundreds of libraries, publishers, merchants and other sources around the globe to compile a vast collection of unique book data searchable by ISBN, title, author, or publisher.
